I wrote a little instruction on where to find the question that needed to be answered, that I wanted the answer printed, and please don’t use any libraries.

Then I just pasted the whole question in full to the GPT chat.

And lo and behold, GPT produced the following program

code 
# Read the input file and split it into lines
with open("input.txt") as f:
  lines = f.read().strip().split("\n")

# Initialize variables to store the maximum Calories and the current Elf's Calories
max_calories = 0
current_calories = 0

# Loop over the lines in the input
for line in lines:
  # If the line is empty, update the maximum Calories and reset the current Elf's Calories
  if not line:
    max_calories = max(max_calories, current_calories)
    current_calories = 0
  else:
    # Otherwise, add the Calories from the current line to the current Elf's Calories
    current_calories += int(line)

# Update the maximum Calories one final time
max_calories = max(max_calories, current_calories)

# Print the result
print(max_calories)

# This program reads the input file line by line, and keeps track of the maximum number of Calories and the current Elf's Calories. Whenever it encounters an empty line, it updates the maximum Calories and resets the current Elf's Calories. At the end, it prints the final value of the maximum Calories, which is the answer to the problem.

I copied the code and ran it on my machine, it produced a number!… I pasted the number into Advent Of Code as the answer and it was the correct answer! Wow…

This prompt that i gave created this code

Write a python program that consists of two parsers, one for stacks and one for moves. 

Make sure to represent the concepts. 
Move, Stack, and Block as classes in the code. 

Move should have methods
getFrom, getTo, getAmount

Stack should have the regular operations that a stack datastructure has.
pop, push, peek, plus id.

Block only needs, getValue which is the character that was read

The data is given in two parts separated by a blank row. 
The first part gives all the block information and the second part all the moves. 

After finding the part of the input that contains the blockpart do the following. 
1. scan the last line of this input
2. for each integer you find call a function create_stack
3. create_stack will need to use the offset from the left and a index which is the integer that was found while scanning. 
4. create_stack will use this offset and look at each previous line at that offset and read in the one letter character that is there, and make a block out of that containing that character. That block will be put on the stack. when there is no more lines to read from (going upwards) that stack is done. proceed to keep scanning until no more integers can be found. There does not have to be any blocks above the integer then that stack has no blocks.

The second parser will create Move objects. 

The input that comes after the block part + the empty line.
STEPS
1. filter out all characters that are not integers in the input.
2. From the remaining integers, the first integer is the amount, the second one is the from, and the third one is the to. 
3. create a Move object.


Now write a main function that
That reads a file called input.txt, runs the parsers on the input then goes on todo the following.

for each Move in order
1. find the Stack with the index/id of Move.getFrom()
2. find the Stack with index/id of Move.getTo()
3. pop Move.getAmount() number of Block from the 'from' Stack.
4. push those blocks that was just popped onto the 'to' Stack. 

After all these operations have been done. 

Go through all Stacks in order, lowest to highest id. 
and print whatever is on the top of each Stack to the terminal. 
 
Make sure the program calls main()
import re

class Block:
  def init(self, value):
  self.value = value

  def getValue(self):
    return self.value

class Stack:
  def init(self, id):
  self.stack = []
  self.id = id

  def push(self, block):
      self.stack.append(block)

  def pop(self, amount):
      popped_blocks = []
      for i in range(amount):
          popped_blocks.append(self.stack.pop())
      return popped_blocks

  def peek(self):
      return self.stack[-1]

  def getId(self):
      return self.id
class Move:
  def init(self, from_stack, to_stack, amount):
  self.from_stack = from_stack
  self.to_stack = to_stack
  self.amount = amount

def getFrom(self):
    return self.from_stack

def getTo(self):
    return self.to_stack

def getAmount(self):
    return self.amount

def create_stack(offset, index):
  stack = Stack(index)
  for i in range(offset, -1, -1):
  block_value = input_lines[i][offset]
  block = Block(block_value)
  stack.push(block)
  return stack

def parse_blocks():
  stacks = []
  block_part = input_lines[:input_lines.index('')]
  last_line = block_part[-1]
  offsets = [m.start() for m in re.finditer(r'\d', last_line)]
  for offset in offsets:
    index = int(last_line[offset])
    stack = create_stack(offset, index)
    stacks.append(stack)
  return stacks

def parse_moves():
  moves = []
  move_part = input_lines[input_lines.index('')+1:]
  for line in move_part:
    move_parts = line.split()
    amount = int(move_parts[0])
    from_stack = int(move_parts[1])
    to_stack = int(move_parts[2])
    move = Move(from_stack, to_stack, amount)
    moves.append(move)
  return moves

def main():
  global input_lines
  with open('input.txt') as f:
    input_lines = f.read().splitlines()

  stacks = parse_blocks()
  moves = parse_moves()

for move in moves:
    from_stack = [stack for stack in stacks if stack.getId() == move.getFrom()][0]
    to_stack = [stack for stack in stacks if stack.getId() == move.getTo()][0]
    blocks = from_stack.pop(move.getAmount())
    for block in blocks:
        to_stack.push(block)

for stack in stacks:
    print(stack.peek().getValue())
main()
